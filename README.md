# 🎓 Repositorio Maestro: Patrones de Diseño en Python

¡Bienvenido a este repositorio educativo integral! Aquí encontrarás una colección detallada de los patrones de diseño más importantes, organizados en volúmenes para facilitar tu aprendizaje progresivo.

Cada ejemplo está implementado con **Python 3.10+**, utilizando *Type Hinting*, Programación Orientada a Objetos (POO) y comentarios detallados en español.

---

## 📂 Índice de Contenidos

### [Volumen 1: Patrones Fundamentales](./)
1.  **[Singleton](./01_Singleton/database_config.py)**: Garantiza una única instancia de una clase.
2.  **[Factory Method](./02_Factory_Method/notifications.py)**: Delega la creación de objetos a subclases o fábricas.
3.  **[Observer](./03_Observer/newsletter.py)**: Notifica cambios a múltiples observadores.
4.  **[Strategy](./04_Strategy/discounts.py)**: Permite intercambiar algoritmos en tiempo de ejecución.
5.  **[Decorator](./05_Decorator/coffee_shop.py)**: Añade funcionalidades a objetos dinámicamente.

### [Volumen 2: Estructura y Comportamiento Avanzado](./Volumen_2/)
1.  **[Adapter](./Volumen_2/01_Adapter/payment_adapter.py)**: Hace compatibles interfaces que antes no lo eran.
2.  **[Facade](./Volumen_2/02_Facade/home_theater.py)**: Simplifica el acceso a sistemas complejos.
3.  **[Command](./Volumen_2/03_Command/smart_bulb.py)**: Encapsula peticiones como objetos independientes.
4.  **[Proxy](./Volumen_2/04_Proxy/secure_document.py)**: Controla el acceso a objetos sensibles.
5.  **[State](./Volumen_2/05_State/traffic_light.py)**: Cambia el comportamiento basado en el estado interno.

---

## 🚀 Cómo usar este repositorio

Para ejecutar cualquier ejemplo, navega hasta la carpeta correspondiente y utiliza Python desde tu terminal:

```bash
# Ejemplo: Ejecutar el patrón Adapter del Volumen 2
python Volumen_2/01_Adapter/payment_adapter.py
```
